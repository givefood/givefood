# Year Summaries

This directory contains yearly summaries of the main code changes to the Give Food project. Each file provides a chronological overview of significant commits organized by month, helping track the evolution of the codebase over time.

The summaries focus on substantive changes such as new features, bug fixes, API updates, and major refactoring work, while filtering out minor changes like typos and whitespace adjustments.
